# POLÍTICA DE PRIVACIDADE — ENVIO COMPROVADO

**Última atualização: 09 de maio de 2026**

---

## 1. INTRODUÇÃO

O Envio Comprovado ("nós", "nosso" ou "aplicativo") é um software desenvolvido para auxiliar vendedores de marketplaces a registrarem o processo de embalagem de pedidos. Esta Política de Privacidade descreve como tratamos as informações no âmbito do uso do aplicativo.

---

## 2. DADOS QUE NÃO COLETAMOS

O Envio Comprovado **não coleta, não armazena e não transmite** nenhum dado pessoal para servidores externos ou terceiros.

Especificamente:
- Não coletamos nome, e-mail, CPF ou qualquer dado de identificação
- Não criamos contas de usuário
- Não enviamos dados para a internet
- Não utilizamos serviços de análise (analytics)
- Não exibimos anúncios
- Não rastreamos a localização do dispositivo
- Não acessamos a lista de contatos, calendário ou outros apps

---

## 3. DADOS PROCESSADOS LOCALMENTE

O aplicativo utiliza as seguintes funcionalidades do dispositivo:

**Câmera e Microfone**
Utilizados exclusivamente para gravar vídeos do processo de embalagem. As gravações são armazenadas **apenas no computador do usuário**, dentro da rede Wi-Fi local. Nenhuma gravação é enviada para servidores externos.

**Rede Wi-Fi local**
O app se comunica com o software instalado no computador do usuário através da rede Wi-Fi local (rede interna). Esta comunicação ocorre exclusivamente dentro da rede privada do usuário e **nunca sai para a internet**.

**Armazenamento local**
As configurações do app (como o endereço IP do computador) são armazenadas localmente no dispositivo, de forma criptografada (AES-256 via Keychain do sistema operacional).

---

## 4. COMUNICAÇÃO ENTRE DISPOSITIVOS

O Envio Comprovado funciona em modo completamente local:

- O celular envia os vídeos diretamente ao computador do usuário via Wi-Fi
- Nenhum dado passa por servidores intermediários
- Os vídeos são salvos na pasta escolhida pelo usuário no computador
- O usuário tem controle total sobre os arquivos gerados

---

## 5. PERMISSÕES DO SISTEMA

| Permissão | Finalidade |
|-----------|-----------|
| Câmera | Gravar vídeos do processo de embalagem |
| Microfone | Incluir áudio nas gravações |
| Wi-Fi / Rede | Enviar vídeos ao computador na rede local |
| Manter tela ativa | Evitar que a tela apague durante a gravação |

Nenhuma permissão é usada para finalidade diferente da descrita acima.

---

## 6. SEGURANÇA

Adotamos as seguintes medidas de segurança:

- Configurações armazenadas com criptografia AES-256 (Keychain do SO)
- Comunicação autenticada por token de sessão único gerado a cada uso
- Nenhum dado sensível trafega pela internet
- Logs de diagnóstico armazenados localmente com limite de 50KB

---

## 7. MENORES DE IDADE

O Envio Comprovado não é direcionado a menores de 18 anos e não coleta dados de menores. É um aplicativo profissional para uso comercial.

---

## 8. ALTERAÇÕES NESTA POLÍTICA

Caso esta política seja alterada, notificaremos os usuários por meio de atualização do aplicativo. A data da última atualização está sempre indicada no topo deste documento.
